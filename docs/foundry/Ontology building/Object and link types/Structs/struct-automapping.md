# [](#automapping-struct-properties)Automapping struct properties结构属性自动映射


Automapping allows users to map all columns automatically rather than manually.自动映射允许用户自动映射所有列，而非手动。


## [](#automap-struct-types-in-ontology-manager)Automap struct types in Ontology ManagerOntology Manager 中的 Automap struct 类型


If the object has already been created, users can automap all columns by using the **Automap all** feature.如果对象已经创建好，用户可以通过自动映射功能对所有列进行自动映射 。


1. In Ontology Manager, enter the **Properties** tab and select the desired property.在 Ontology Manager 中，进入属性标签并选择所需的属性。
2. Under the **Column mapping** tab, select the desired column.在“ 列映射 ”标签下，选择所需的列。


![The 'Column mapping' tab and the 'Automap all' button.](automap-struct-oma.png?width=500)


1. Select **Automap all**.选择全部自动地图 。


## [](#automap-struct-types-in-pipeline-builder)Automap struct types in Pipeline Builder管道构建器中的自动映射结构类型


If the object has not yet been created, automapping can be done on initial object creation with the object type creation wizard.如果对象尚未被创建，可以在初始创建对象时使用对象类型创建向导进行自动映射。


1. In your Pipeline Builder pipeline, open the relevant dataset and select the **All Actions** dropdown in the top right.在你的管道构建器流水线中，打开相关数据集，选择右上角的“所有动作 ”下拉菜单。


![The All actions dropdown in the dataset detail page.](automap-struct-pipelinebuilder.png?width=500)


1. Select **Create object type** to create a new object.选择创建对象类型以创建新对象。


![The Properties tab in the 'Create a new object' dialog.](automap-struct-properties.png?width=500)


1. Under **Properties**, add the desired properties to be mapped.在属性中，添加想要映射的属性。
2. Select  and complete the remaining steps to create an automapped object type.选择 “下一步 ”，完成剩余步骤以创建自动映射对象类型。
