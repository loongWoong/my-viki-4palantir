# [](#create-a-value-type)Create a value type创建一个值类型


Follow the steps below to create a value type to use across your platform [space](/docs/foundry/security/orgs-and-spaces/#spaces).请按照以下步骤创建适用于整个平台空间的价值类型。


1. Navigate to the **Value Types Manager** application from the platform sidebar.从平台侧边栏导航到价值类型管理器应用。
2. From the top left corner, use the dropdown menu to select the space in which you would like to create a value type.在左上角，使用下拉菜单选择你想创建值类型的空间。
3. Select **Create New Value Type** from the upper right corner.从右上角选择创建新值类型 。
4. Provide a clear name, description, and unique API name for your value type.为你的值类型提供清晰的名称、描述和唯一的 API 名称。


![Value type metadata creation](value-type-create-metadata.png?width=500)


1. Choose a [base type](/docs/foundry/object-link-types/base-types/) for your value type.为你的价值类型选择一个基础类型 。
2. (Optional) Define a constraint for your value type. Validators can be regular expressions for `String` types, enums, ranges, or other validation methods depending on the base type.
For a full list of constraints supported by base type, review our [value type constraints](/docs/foundry/object-link-types/value-type-constraints/) documentation.（可选）为你的值类型定义一个约束。验证器可以是字符串类型、枚举、范围或其他验证方法的正则表达式，具体取决于基类型。有关基础类型支持的完整约束列表，请查看我们的值类型约束文档。


![Value type constraint creation](value-type-create-constraint.png?width=500)


1. (Optional but recommended) Provide an example preview value for your value type.（可选但推荐）提供你的值类型的示例预览值。


![Value type preview creation](value-type-create-preview.png?width=500)


1. Save your value type.保存你的价值类型。
